# E2E Testing Harness Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up a local, deterministic end-to-end test (Playwright driving the real React frontend + real Go services via `wails dev`) for the core chat smoke chain, with the agent runtime replaced by a build-tag-gated fake so no subprocess/token/login is needed.

**Architecture:** Playwright's own chromium hits `http://localhost:34115` (the `wails dev` server, which serves the real frontend *with* Go bindings). The whole backend runs for real against an isolated throwaway SQLite DB (`AGENTRE_DATA_DIR` → temp dir). Only `agentruntime.RuntimeFor(claudecode)` is swapped for a deterministic fake, registered + seeded by an installer compiled only under `-tags e2e`. Production builds compile a no-op installer and never include the fake.

**Tech Stack:** Go 1.26, Wails v2.12.0, React 19/TS, Vite, Playwright (`@playwright/test`), pnpm 10.33. Go test stack: testify + goconvey. Build tag: `e2e`.

**Key facts pinned during design (do not re-derive):**
- `agentruntime.Runtime` interface = `Capabilities() capability.Capabilities` + `Run(ctx, RunRequest) (<-chan Event, *RunResult, error)` (`internal/pkg/agentruntime/registry.go:19-27`).
- Events are plain structs emitted on the channel: `agentruntime.TextDelta{Text}`, `agentruntime.Done{}` (`internal/pkg/agentruntime/event.go`).
- Override registration: `agentruntime.RegisterRuntime(agent_backend_entity.TypeClaudeCode, impl)` (sets the registry map; later call wins).
- A **local claudecode backend with empty `LLMProviderKey` is valid** (CLI-login mode) — `claudeCodeKind.ValidateExtra` allows it (`internal/model/entity/agent_backend_entity/kinds.go:119-128`).
- The default agent ("CEO 助手", `system_badge=DEFAULT`) is seeded with `agent_backend_id=0` and is fetched badge-based via `agent_repo.Agent().FindSystem(ctx)` (`internal/repository/agent_repo/agent.go:80`). No department is seeded.
- `agent_svc.Agent().Update` allows attaching a backend to the CEO agent (`internal/service/agent_svc/agent.go:96-130`); `agent_backend_svc.AgentBackend().Create` returns `resp.Item.ID` (`internal/service/agent_backend_svc/agent_backend.go:105-150`).
- `wails dev` serves at `http://localhost:34115` and accepts `-tags e2e`. The rendered `aria-label`s are `t(...)` **translated strings**, not i18n keys — use `data-testid`/`role` selectors, never `[aria-label="<key>"]`.

```go
//go:build e2e

// Package fake 提供 e2e 专用的确定性 agent runtime:不起任何子进程,按 req.UserText
// 回显一段固定前缀文本后正常结束。仅在 `-tags e2e` 构建中编译,生产二进制不含本包。
package fake

import (
	"context"
	"fmt"

	"agentre/internal/pkg/agentruntime"
	"agentre/internal/pkg/agentruntime/capability"
)

// ReplyPrefix 是所有假回复的前缀,前端据此断言并与用户消息区分。
const ReplyPrefix = "e2e-fake-reply: "

// Runtime 实现 agentruntime.Runtime。
type Runtime struct{}

// New 返回一个 fake runtime。
func New() *Runtime { return &Runtime{} }

// Capabilities 返回一个最小能力集(仅支持中止),足够驱动聊天 UI。
func (r *Runtime) Capabilities() capability.Capabilities {
	return capability.Capabilities{
		Set: map[capability.Capability]bool{
			capability.CapAbort: true,
		},
	}
}

// Run 把 ReplyPrefix+UserText 分片流式发送后 emit Done。
func (r *Runtime) Run(ctx context.Context, req agentruntime.RunRequest) (<-chan agentruntime.Event, *agentruntime.RunResult, error) {
	out := make(chan agentruntime.Event, 8)
	result := &agentruntime.RunResult{
		ProviderSessionID: fmt.Sprintf("e2e-fake-%d", req.SessionID),
		Model:             "e2e-fake-model",
	}
	reply := ReplyPrefix + req.UserText
	go func() {
		defer close(out)
		for _, chunk := range splitChunks(reply, 8) {
			select {
			case <-ctx.Done():
				return
			case out <- agentruntime.TextDelta{Text: chunk}:
			}
		}
		out <- agentruntime.Done{}
	}()
	return out, result, nil
}

// splitChunks 按 rune 边界把 s 切成最多 n 个 rune 的片段。
func splitChunks(s string, n int) []string {
	if n <= 0 || s == "" {
		return []string{s}
	}
	runes := []rune(s)
	out := make([]string, 0, (len(runes)+n-1)/n)
	for i := 0; i < len(runes); i += n {
		end := i + n
		if end > len(runes) {
			end = len(runes)
		}
		out = append(out, string(runes[i:end]))
	}
	return out
}
```

```go
//go:build e2e

package main

import (
	"context"

	"github.com/cago-frame/cago/pkg/logger"
	"go.uber.org/zap"

	"agentre/internal/model/entity/agent_backend_entity"
	"agentre/internal/pkg/agentruntime"
	fakert "agentre/internal/pkg/agentruntime/runtimes/fake"
	"agentre/internal/repository/agent_repo"
	"agentre/internal/service/agent_backend_svc"
	"agentre/internal/service/agent_svc"
)

// installE2EFakes 仅在 `-tags e2e` 构建中编译:
//  1. 用确定性 fake 覆盖 claudecode runtime(无子进程/无登录);
//  2. seed 一个本地 claudecode backend 并挂到默认 CEO agent,
//     让前端"建会话→发消息→看回复"无需真实 CLI 即可跑通。
//
// 失败只记日志不 panic:e2e 环境异常应让 Playwright 用例红,而不是让 app 崩。
func installE2EFakes(ctx context.Context) {
	agentruntime.RegisterRuntime(agent_backend_entity.TypeClaudeCode, fakert.New())

	resp, err := agent_backend_svc.AgentBackend().Create(ctx, &agent_backend_svc.CreateBackendRequest{
		Type: string(agent_backend_entity.TypeClaudeCode),
		Name: "E2E Local Backend",
	})
	if err != nil {
		logger.Ctx(ctx).Error("main.installE2EFakes: create backend failed", zap.Error(err))
		return
	}

	ceo, err := agent_repo.Agent().FindSystem(ctx)
	if err != nil || ceo == nil {
		logger.Ctx(ctx).Error("main.installE2EFakes: find system agent failed", zap.Error(err))
		return
	}

	if _, err := agent_svc.Agent().Update(ctx, &agent_svc.UpdateAgentRequest{
		ID:             ceo.ID,
		Name:           ceo.Name,
		AgentBackendID: resp.Item.ID,
	}); err != nil {
		logger.Ctx(ctx).Error("main.installE2EFakes: attach backend to agent failed", zap.Error(err))
		return
	}

	logger.Ctx(ctx).Info("main.installE2EFakes: e2e fakes installed",
		zap.Int64("backendID", resp.Item.ID), zap.Int64("agentID", ceo.ID))
}
```

```ts
import { existsSync, readFileSync, rmSync } from "node:fs";
import path from "node:path";

// 删除本次运行的一次性数据目录与 sentinel。
export default async function globalTeardown() {
  const sentinel = path.join(__dirname, ".data-dir");
  if (!existsSync(sentinel)) return;
  const dataDir = readFileSync(sentinel, "utf8").trim();
  if (dataDir) rmSync(dataDir, { recursive: true, force: true });
  rmSync(sentinel, { force: true });
}
```

```ts
import { test, expect } from "@playwright/test";

// 核心冒烟链路:打开 app → 新建 agent 会话 → 发消息 → 看到 fake 流式回复 → tab 回到 idle。
// 后端用 -tags e2e 的 fake runtime,回复固定前缀 "e2e-fake-reply: "。
test("core chat smoke chain", async ({ page }) => {
  await page.goto("/");

  // 1. app 加载完成:新建按钮可见。
  const newChat = page.getByTestId("new-chat-button");
  await expect(newChat).toBeVisible();

  // 2. 新建 agent 会话:打开下拉 → 选 "新建 agent 会话" → 选种子里的默认 agent。
  await newChat.click();
  await page.getByTestId("new-agent-chat-item").click();
  await page.locator('[data-testid^="agent-picker-item-"]').first().click();

  // 3. 会话已打开:激活 tab + 输入框就绪。
  await expect(page.locator('[role="tab"][data-active="true"]')).toBeVisible();
  const editor = page.getByRole("textbox");
  await expect(editor).toBeVisible();

  // 4. 输入并发送(ProseMirror 用 pressSequentially)。
  await editor.click();
  await editor.pressSequentially("ping");
  await page.getByTestId("chat-send-button").click();

  // 5. fake 回复出现在转录区(固定前缀,确定性)。
  await expect(page.getByText(/e2e-fake-reply: ping/)).toBeVisible();

  // 6. turn 收尾:tab 不再 running(spinner 消失) —— 守"卡 running"老坑。
  await expect(page.getByTestId("tab-spinner")).toHaveCount(0);
});
```
